Package logging

source code

Logging module for Pyjamas, mimicking CPython's logging module.

Usage example:
    from pyjamas import logging
    log = logging.getConsoleLogger()
    log.debug('This is a debug message')

Author: Peter Bittner <peter.bittner@gmx.net>
