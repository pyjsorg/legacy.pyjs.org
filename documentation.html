Documentation and References

A complete reference guide - the API documentation - is available, which can be built locally using the gendocs.py script, in the Pyjs distribution.

There are also HOWTO Guides at the Pyjs Wiki. These HOWTOs are written by users, for users, and will guide you through the process of setting up and using Pyjs, including with your preferred web server framework (Django, Pylons, Web.py etc.)

Pyjs Book

An online version of the Pyjs Book is available to read, and is in development. Contact the mailing list if there are any issues that you believe should be covered; if you find any errors. The Bookreader itself is a Pyjs application which uses AJAX to load the contents as text files, and Wiki-like markup is used, on the browser, to generate the HTML displayed.

Mailing List

Pyjs' mailing list is pyjs-users. Please use the web interface to join.

Pyjs' former (developer and user) mailing list was pyjamasdev@pyjs.org. You can search for older mails in one of our Mailing List archives (Google Groups, Mail Archive, Nabble, Gmane).
